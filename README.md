# HTML & CSS Layout Design

## Project Overview

This project is a simple webpage layout created using **HTML5** and **CSS3**. The purpose of this project is to recreate a given layout design by using basic web development concepts such as containers, divisions, colors, borders, spacing, and Flexbox.

The webpage consists of six colored sections arranged in a structured format. The design demonstrates how different HTML elements can be organized and styled to match a reference layout while maintaining clean and readable code.

---

# Objectives

- Learn the basic structure of an HTML webpage.
- Practice CSS styling techniques.
- Understand the use of CSS Flexbox for layouts.
- Apply borders, spacing, margins, and padding.
- Recreate a given design using only HTML and CSS.

#  Project Components

The webpage contains the following sections:

###  Header
- Positioned at the top.
- Red background color.
- Full-width container.

### Navigation Section
- Located below the header.
- Blue background.
- Full-width layout.

###  Left Content Section
- Yellow rectangular box.
- Positioned on the left side.

### Lower Left Section
- Orange rectangular box.
- Placed directly below the yellow section.

###  Right Sidebar
- Pink vertical box.
- Positioned beside the yellow and orange boxes.

###  Footer
- Gray background.
- Full-width footer section.

# How It Works

1. The **HTML** file creates the webpage structure.
2. The **CSS** file styles each section.
3. Flexbox is used to arrange the middle content.
4. Borders and spacing create the visual layout.
5. Colors distinguish different webpage sections.
